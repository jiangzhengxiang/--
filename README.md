# 今日无事 (NO AFFAIRS) - 内在觉察与释放助手

一个基于 **Vue 3** + **Vite** + **Tailwind CSS** 构建的沉浸式正念应用，旨在帮助用户通过科学的方法释放情绪阻力。

## 🌟 核心功能

- **呼吸引导 (Breathing Guide)**：提供腹式呼吸与闭气练习，配以动态视觉辅助。
- **释放引导 (Release Guide)**：基于莱斯特·利文森的释放理论，通过交 互 式 问 答引导用户“放下”情绪。
- **AI 情绪助手 (LLM)**：接入 DeepSeek API，支持通过勾选知识库来精准注入专业背景知识，提供个性化引导。
- **智慧知识库 (Knowledge Base)**：收录情绪释放理论、欲望机制、生理放松等核心深度内容。

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **路由**：Vue Router
- **AI 接入**：DeepSeek API (Fetch API)

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/emotion-project.git
   cd emotion-project
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **运行开发服务器**
   ```bash
   npm run dev
   ```

4. **配置 AI**
   在“大模型 -> 配置”中输入您的 DeepSeek API Key 即可开启智能对话。

## 📸 预览

项目包含精美的书法风格纵向排版，采用宁静的绿色调，支持深色模式。
